Popular Girl Names API

This project ran through the API linked and chose data from the API based on user input. We were asked to link an API to our program and make it go through the API. The user is prompted to enter a name. The result given to the user is that they tell you the rank of the name on the list of APIs based on how many girls have the same name in New York.

A challege my group faced was a challenge through the projects extension. We wanted to make the project ask the user if they wanted a name suggestion. People can say "Yes" or "No" and depending on the response the program will run through the API and randomly pick a API name. The challenge was getting it to randomly pick a name. We fixed this by asking the user to pick a number from 1-1000 and that picked the baby name given to them. Next steps to take on this project is to edit the site to look nicer. I would also fix the wording on some of the prompts to be more specific and user friendly so the program runs smoother.
